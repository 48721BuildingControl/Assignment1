Electrical Lighting System

  • Used only when daylight isn't sufficient
  • Primarily energy efficient lights like LED and fluorescent
  • Occupancy sensors turn off lights when no one is using them
  • Dimming ballasts ensure only enough electricity as is necessary is used
  • Lighting control system ensures lights used effectively
  • Making own electricity offsets electricity usage to make the building zero net energy

General Operating and Control

  • Building Management System (BMS) used to monitor energy use
  • Web interface for monitoring conditions and equipment
  • Sensors detect daylight and occupancy and change lights to conserve electricity
  • Occupants can override system when necessary
  • System can increase energy efficiency

Lighting Arrangement Programming

  • Two SpecGrade Dimming Panels allow programming of lighting arrangements in several main rooms
  • Arrangements are specific sets of lights in specific rooms to best accomodate a given use of the room
  • Different lights can be used for having a conference versus a presentation
  • Allows most efficient lighting arrangements to be pre-determined
  • Arrangements from dimming panels are linked to seeTouch wall stations in the rooms
  • 2-4 potential arrangements each
  • Occupants tap button for desired scene and lights change
